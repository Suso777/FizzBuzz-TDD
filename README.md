# FizzBuzz-TDD

¡Bienvenido al proyecto FizzBuzz-TDD! Este proyecto es una implementación del clásico juego FizzBuzz, usando un enfoque de Desarrollo Basado en Pruebas (TDD).

### ¿Qué es FizzBuzz?

FizzBuzz es un juego comúnmente utilizado en entrevistas de programación para evaluar las habilidades de codificación de los desarrolladores. La regla es simple: contar números desde 1 hasta un número dado y reemplazar los múltiples de 3 con "Fizz", los múltiplos de 5 con "Buzz", y los múltiplos de ambos con "FizzBuzz".

### Enfoque TDD

Este proyecto utiliza TDD para garantizar que todas las funcionalidades estén bien probadas y sean confiables desde el principio. Al escribir las pruebas primero, podemos desarrollar el código necesario para que pasen, lo que conduce a una base de código más sólida y fácil de mantener.
